About This Project

Trade Tracker Hub is a web-based trading dashboard designed to help users monitor stock prices, visualize live market movements, and manage their portfolio in a simple and user-friendly way.

This project is created as a learning-based application using HTML, CSS, JavaScript, and APIs to simulate real-time stock market behavior.

Purpose of the Project

The main purpose of this project is to understand how real-time data visualization, frontend-backend communication, and dashboard-based applications work. It is ideal for students and beginners learning web development and financial applications.
